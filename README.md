# 💻 Programación – CFGS Desarrollo de Aplicaciones Multiplataforma (DAM)

## 📍 Andalucía
Repositorio dedicado a prácticas, ejercicios y proyectos del módulo de **Programación** del ciclo formativo de grado superior **Desarrollo de Aplicaciones Multiplataforma (DAM)**.

---

## 📌 Descripción

En este repositorio se recogen los ejercicios y proyectos realizados durante el módulo de Programación, enfocados al aprendizaje de:

- Programación orientada a objetos (POO)
- Estructuras de control
- Métodos y funciones
- Manejo de excepciones
- Colecciones
- Entrada y salida de datos
- Buenas prácticas y modularización del código

---

## 🛠 Tecnologías utilizadas

- Java
- IntelliJ / Eclipse
- Git y GitHub

---

## 🎯 Objetivos

- Desarrollar lógica de programación.
- Aplicar principios de programación orientada a objetos.
- Escribir código limpio y estructurado.
- Preparar base sólida para módulos posteriores (Acceso a Datos, Programación de Servicios y Procesos, etc.).
